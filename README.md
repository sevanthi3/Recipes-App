# 🥘 Recipes CRUD App

A full-featured backend API to manage recipes using **Node.js**, **Express.js**, **MongoDB**, and **Mongoose**. It follows the **MVC architecture** and uses **Postman** for API documentation and testing.

## 🚀 Tech Stack Used

- ⚙️ Node.js  
- 🚦 Express.js  
- 🍃 MongoDB  
- 🧬 Mongoose  
- 📬 Postman
